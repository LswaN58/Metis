Quick Reference for JavaScript Libraries:

There are many JavaScript libraries in the world. This page attempts to document some of them. This page will not try to convince you which library to use. It will not try to teach you any given library in depth (follow the link(s) for that). Instead, this page will present a quick reference for each library, including:

  • Library name and most recent version tested
  • A link to library homepage
  • A link or script tag to link the library, typically via CDN.
    Where possible, I've chosen a CDN from the library producer, rather than a 3rd-party CDN service.
  • A bare-bones example that can be easily copied and modified.
  • Where applicable, brief tips to ensure the library is properly used/initialized.
  • Where applicable, a short reference table of common functions and their uses.
