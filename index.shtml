Quick Reference for HTML and JavaScript Libraries:

There are many HTML/CSS/JS libraries in the world. This website attempts to document some of them. This site will not try to convince you which libraries to use. It will not try to teach you any given library in depth (follow the link(s) for that). Instead, this site will present a quick reference for each library, as well as a number of handy built-in HTML/CSS/JS things.
This homepage documents very basic concepts such as file inclusion and some lesser-known built-in topics. Other pages cover various categories of library.

HTML Lesser-known built-ins:

  • UTF characters can be hand-added to html. This is not super-convenient, but useful in cases.
    Three options for syntax:

    1. &#num; , where num is the utf-8 number
    2. &#xnum; , where num is the hexadecimal form of the utf-8 number
    3. &name; , where name is the html entity name for the symbol. Not all UTF characters have entity names.
