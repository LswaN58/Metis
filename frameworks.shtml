Quick Reference for JavaScript Frameworks:

There are many JavaScript frameworks in the world. This page attempts to document some of them. This page will not try to convince you which framework to use. It will not try to teach you any given framework in depth (follow the link(s) for that). Instead, this page will present a quick reference for each framework, including:

  • Framework name and most recent version tested
  • A link to framework homepage
  • Brief outline of code to ensure the framework is properly used/initialized.
